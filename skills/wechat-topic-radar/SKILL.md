# 公众号蓝海选题雷达

帮助公众号创作者找到蓝海选题方向，从数据调研到运营全案一站式输出。

## 触发条件

当用户需要：
- 公众号选题方向分析
- 找蓝海赛道/避开红海
- 内容定位建议
- 公众号运营方案

## 核心流程

```
输入类目+定位 → 全网数据扫描（蓝海报告）→ 二轮深度筛选（精选选题）→ 定稿输出（运营全案）
```

---

## 第一步：全网数据扫描

### 目标
判断一个类目是蓝海还是红海

### 蓝海判定公式
```
蓝海指数 = 搜索热度 × 增长趋势 ÷ 竞争强度

- 搜索量高 + 竞争度低 + 增长趋势上升 = 🏆 蓝海金矿
- 搜索量高 + 竞争度高 + 趋势平稳 = ⚠️ 红海市场
- 搜索量低 + 竞争度低 = ❓ 需求不足
```

### 数据来源

| 平台 | 数据类型 | 获取方式 |
|------|----------|----------|
| 微信搜一搜 | 搜索热度、相关词 | web_search 模拟 |
| 新榜 | 公众号排行、爆文数据 | web_fetch 抓取 |
| 5118 | 关键词竞争度、长尾词 | web_fetch 抓取 |
| 百度指数 | 搜索趋势、人群画像 | web_search 查询 |
| 微信指数 | 微信生态热度 | web_search 查询 |

### 执行 Prompt

```
你是一个公众号选题分析专家。用户想做【{类目}】方向的公众号，定位是【{定位}】。

请帮我做一份蓝海选题调研报告：

1. **市场热度分析**
   - 搜索这个类目相关的关键词热度
   - 分析近期趋势（上升/平稳/下降）

2. **竞争格局分析**
   - 搜索该类目头部公众号数量和粉丝量级
   - 分析内容同质化程度

3. **细分方向挖掘**
   - 列出 10 个细分选题方向
   - 每个方向给出蓝海指数评分（1-10）
   - 标注：🟢 蓝海（7-10分）/ 🟡 中等（4-6分）/ 🔴 红海（1-3分）

4. **数据依据**
   - 每个判断都要有数据支撑
   - 引用搜索结果链接

输出格式：Markdown 表格 + 分析说明
```

### 输出示例

```markdown
## 📊 公众号蓝海选题报告

**类目：** AI 工具教程
**定位：** 小白友好的 AI 工具使用指南

### 🎯 细分方向蓝海指数

| 方向 | 搜索热度 | 竞争度 | 趋势 | 蓝海指数 | 判定 |
|------|----------|--------|------|----------|------|
| AI 绘画入门 | 高 | 高 | 平稳 | 4 | 🔴 红海 |
| AI 视频制作 | 高 | 中 | 上升 | 7 | 🟢 蓝海 |
| AI 办公提效 | 中 | 低 | 上升 | 8 | 🟢 蓝海 |
| AI 编程辅助 | 中 | 高 | 平稳 | 3 | 🔴 红海 |
| AI 音乐创作 | 低 | 低 | 上升 | 6 | 🟡 中等 |

### 💡 推荐方向
1. **AI 视频制作** - Seedance/Sora 等工具刚火，教程缺口大
2. **AI 办公提效** - 职场刚需，受众广，变现路径清晰
```

## 第二步：二轮深度筛选

### 目标
从初筛的蓝海方向中，挑出最适合用户的 3 个金矿

### 输入
- 用户从第一步选择感兴趣的方向（可多选）
- 用户补充信息：擅长领域、目标人群、变现预期

### 执行 Prompt

```
用户对以下蓝海方向感兴趣：【{选中的方向}】

用户补充信息：
- 擅长领域：【{擅长}】
- 目标人群：【{人群}】
- 变现预期：【{变现}】

请对每个方向做深度分析：

1. **竞品账号分析**
   - 搜索该方向 TOP5 公众号
   - 分析粉丝量、更新频率、爆文数据

2. **爆文案例拆解**
   - 找 3 篇该方向的 10w+ 爆文
   - 分析标题套路、内容结构、引流钩子

3. **用户画像**
   - 目标用户是谁
   - 痛点是什么
   - 付费意愿如何

4. **变现路径**
   - 流量主收入预估
   - 知识付费可能性
   - 带货/接广告空间

5. **匹配度评估**
   - 和用户擅长领域的匹配度
   - 启动难度（1-5星）
   - 综合推荐指数（1-10）

输出格式：每个方向一张分析卡片
```

### 输出示例

```markdown
## 🔍 深度筛选报告

### 方向 1：AI 视频制作教程

**竞品分析**
| 账号 | 粉丝 | 更新频率 | 爆文率 |
|------|------|----------|--------|
| AI视频工坊 | 12w | 日更 | 15% |
| 剪映教程 | 50w | 周更3篇 | 8% |

**爆文案例**
1. 《Seedance 2.0 保姆级教程，小白也能做电影》- 12w 阅读
2. 《用 AI 做出来的视频，甲方以为我请了团队》- 8w 阅读

**变现路径**
- 流量主：预估 3000-8000 元/月（10w粉丝）
- 知识付费：AI 视频课程，定价 199-499 元
- 接广告：AI 工具推广，单条 2000-5000 元

**匹配度评估**
- 与用户擅长领域匹配度：⭐⭐⭐⭐⭐
- 启动难度：⭐⭐⭐
- 综合推荐指数：**9/10** 🔥

---
```

## 第三步：定稿输出运营全案

### 目标
把选定的方向变成可执行的运营方案

### 输入
用户从第二步选定 1-3 个方向

### 执行 Prompt

```
用户最终选定方向：【{选定方向}】

请生成一份完整的公众号运营全案：

## 一、账号定位
- 账号名称建议（3个备选）
- 一句话定位
- 差异化价值主张

## 二、人设设计
- 人设类型（专家型/朋友型/IP型）
- 人设关键词 5 个
- 自我介绍模板

## 三、内容规划
- 内容类型占比（干货/案例/热点/互动）
- 选题库（30个选题）
- 30天内容排期表

## 四、对标账号拆解
- 3个对标账号详细分析
- 可复用的内容模式
- 差异化切入点

## 五、涨粉策略
- 冷启动方案（0-1000粉）
- 增长期方案（1000-1w粉）
- 爆发期方案（1w-10w粉）

## 六、变现设计
- 变现时间节点
- 变现产品矩阵
- 收入预估

## 七、启动计划
- 第1周：账号搭建 + 首批内容
- 第2-4周：内容测试 + 数据复盘
- 第2-3月：优化迭代 + 涨粉加速

输出格式：结构化 Markdown，可直接用作执行手册
```

## 注意事项

1. **数据时效性**：公众号生态变化快，建议每月更新一次分析
2. **主观判断**：蓝海指数是参考值，最终决策还需结合自身情况
3. **执行为王**：再好的选题不执行也是零，先干起来再优化
