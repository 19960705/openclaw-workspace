# TikTok Thailand Translator v2.0

混合翻译模式：关键词映射 + AI 智能翻译，用于 TikTok 电商文案生成。

## 🚀 快速开始

### 基础翻译（单个文本）

```typescript
POST /api/translate
{
  "action": "translate",
  "text": "这款连衣裙很划算，质量好，发货快"
}
```

响应：
```json
{
  "success": true,
  "original": "这款连衣裙很划算，质量好，发货快",
  "translated": "เดรสนี้คุ้มมาก คุณภาพดี ส่งไว ค่ะ",
  "mode": "keyword",
  "keywords": ["คุ้มมาก", "คุณภาพดี", "ส่งไว"],
  "confidence": 0.8
}
```

### 批量翻译

```typescript
POST /api/translate
{
  "action": "batch",
  "texts": ["买一送一", "限时优惠", "正品保证"],
  "mode": "mixed"
}
```

## 📊 翻译模式

### 1. 关键词模式 (keyword)
- **速度**: ⚡⚡⚡ 最快
- **成本**: 💰 零成本
- **准确度**: 🎯 80%
- **适用场景**: 短文本、包含大量已知关键词

### 2. AI 模式 (ai)
- **速度**: ⚡⚡ 中等
- **成本**: 💰💰 有成本（API 调用）
- **准确度**: 🎯🎯🎯 95%
- **适用场景**: 长文本、复杂句式、产品详情

### 3. 混合模式 (mixed) ⭐ 推荐
- **速度**: ⚡⚡ 快速
- **成本**: 💰 低成本（关键词为主，少量 AI 润色）
- **准确度**: 🎯🎯 85%
- **适用场景**: 产品描述、TikTok 文案、电商内容

## 📦 核心功能

### 1. 翻译服务 (`index.ts`)
- 关键词映射库（100+ 常用词汇）
- 自动模式检测（智能选择最佳翻译模式）
- 批量翻译支持
- 关键词提取

### 2. 1688 爬虫 (`1688-scraper.ts`)
- 产品标题解析
- 价格范围提取
- 规格参数提取
- 特点列表提取

### 3. TikTok Shop 生成器 (`tiktok-shop-generator.ts`)
- 类目模板选择
- 自动文案生成
- Hashtag 生成
- CTA（行动号召）优化
