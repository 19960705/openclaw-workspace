# 运镜复刻 & 特效模板

## 适用场景
- 复刻电影运镜
- 复制广告创意
- MV 风格模仿
- 特效转场复刻

---

## 核心概念

Seedance 2.0 可以从参考视频中提取：
- **运镜方式**：推拉摇移、环绕、跟拍等
- **动作节奏**：舞蹈、打斗、运动动作
- **转场效果**：粒子消散、瞳孔穿越、拼图破碎等
- **剪辑节奏**：卡点、快切、速度曲线

## 常见运镜类型及写法

### 希区柯克变焦 (Dolly Zoom)
```
镜头使用希区柯克变焦效果，推进的同时拉远焦距，主体大小不变但背景剧烈压缩/扩张，产生眩晕和不安感。
```

### 360° 环绕镜头
```
镜头以主体为中心进行 360 度环绕拍摄，缓慢/快速旋转一周，展示主体和周围环境的关系。
```

### 一镜到底 (One Take)
```
整段视频为一镜到底拍摄，无剪辑切换。镜头从 [起点] 流畅移动到 [终点]，中间经过 [场景 A]、[场景 B]、[场景 C]，场景过渡自然。
```

### 跟拍 (Tracking Shot)
```
镜头跟随主体移动，保持主体在画面固定位置。
- 侧面跟拍：镜头在主体侧面平行移动
- 背面跟拍：镜头在主体身后跟随
- 正面跟拍：镜头在主体前方后退拍摄
```

### 升降镜头 (Crane Shot)
```
镜头从 [高位/低位] 缓慢 [下降/上升]，展示场景的纵深层次。
```

### 手持镜头感
```
轻微手持抖动感，增加临场感和纪录片质感，不要太稳定。
```

## 特效转场模板

### 瞳孔穿越转场
```
镜头推近至角色眼睛特写，通过瞳孔进入下一个场景，从瞳孔内部视角飞出，进入新的世界。
```

### 粒子消散转场
```
角色/物体逐渐分解成粒子，粒子飘散重组成新的画面/角色。
```

### 快门闪烁转场
```
画面快速闪烁变白/变黑，闪烁过后切换到新场景。
```

### 旋转/漩涡转场
```
画面开始旋转加速，形成漩涡效果，漩涡中心出现新场景，旋转减速后进入新画面。
```

### 物体遮挡转场
```
[物体] 从画面一侧快速划过遮挡整个画面，遮挡物移开后是新场景。
```

## 生成建议

1. **参考视频要精准**：只截取你需要的运镜片段，越短越准
2. **分离不同参考**：动作用一个视频，运镜用另一个，更可控
3. **文字补充细节**：参考视频 + 文字描述结合效果最佳
4. **多次尝试**：复杂运镜可能需要 3-5 次生成挑选
5. **先短后长**：先用 5 秒测试运镜效果，满意后生成长版本
