# 音乐卡点模板

## 适用场景
- 音乐 MV
- 舞蹈视频
- 节奏感广告
- 风光卡点视频
- Vlog 卡点剪辑

---

## 核心概念

Seedance 2.0 可以根据参考音频/视频的节奏，自动让画面切换卡在节拍上。

**关键写法：**
```
根据 @视频/音频 中的画面节奏进行卡点
```

---

## 卡点提示词模板

### 基础卡点（多图 + 音乐）
```
@图片1 @图片2 @图片3 @图片4 @图片5 中的图片根据 @视频1 中的画面节奏进行卡点。
每个节拍切换到下一张图片。
画面过渡流畅，节奏感强。
```

### 人物舞蹈卡点
```
@图片1 的女孩在舞台上跳舞。
```


### 产品动感卡点
```

每个节拍产品旋转/位移/特效变化。
```

## 节奏描述关键词

| 效果 | 写法 |
|------|------|
| 踩准节拍 | 根据节奏进行卡点 |
| 强拍切换 | 每个强拍触发一次切换 |
| 渐强渐弱 | 节奏从慢到快/从快到慢 |
| 速度变化 | 节拍处进行速度曲线变化 |
| 画面抖动 | 节拍处画面轻微震动 |
| 闪烁效果 | 节拍处灯光/颜色闪烁 |

## 音频参考技巧

### 方式 1：使用带音乐的视频作为参考
```
根据 @视频1 中的画面节奏进行卡点
```
模型会自动识别视频中的音乐节奏。

### 方式 2：上传纯音频文件
```
根据 @音频1 的节奏进行卡点
```
直接使用 MP3 文件。

### 方式 3：描述节奏特征
```
视频整体节奏感强，每隔 1 秒进行一次画面切换，配合电子音乐的强节拍。
```
当没有参考素材时使用文字描述。

## 常见卡点类型

### 1. 硬切卡点
每个节拍直接切换画面，干脆利落。
```
每个节拍硬切到下一个画面，无过渡效果。
```

### 2. 闪白卡点
节拍处画面闪白过渡。
```
每个节拍画面闪白一帧后切换。
```

### 3. 速度曲线卡点
节拍处慢动作或加速。
```
强拍处慢动作，弱拍处正常速度。
```

### 4. 变焦卡点
节拍处快速推拉。
```
每个节拍镜头快速推进或后拉。
```

### 5. 抖动卡点
节拍处画面震动。
```
每个强拍画面轻微抖动，增加冲击感。
```

## 素材准备建议

| 素材类型 | 数量建议 | 说明 |
|----------|----------|------|
| 图片 | 5-9 张 | 按想要的出场顺序排列 |
| 音乐参考视频 | 1 个 | 节奏清晰的短视频 |
| 音频（可选） | 1 个 | MP3 格式，≤15s |

### 图片选择技巧
- 风格统一：色调、光线、构图保持一致
- 内容多样：不同角度、不同元素增加变化
- 按顺序命名：图片 1 → 图片 2 → ... 对应出场顺序

---

## 生成建议

1. **音乐节奏要清晰**：选择节拍明显的音乐作为参考
2. **图片数量匹配节拍**：15 秒视频约 8-15 个节拍，准备相应数量的图片
3. **先听后做**：先听参考音乐，数清节拍数，再准备对应数量的素材
4. **试生成 5 秒**：先用 5 秒版本测试卡点效果
5. **多次尝试**：卡点有一定随机性，多生成几次挑最准的
