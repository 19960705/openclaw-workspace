# Seedance 2.0 Prompt Generator

为 Seedance 2.0（字节跳动即梦 AI 视频模型）生成专业级提示词。

## 触发条件

当用户需要：
- 生成 Seedance 2.0 视频提示词
- 创建分镜头脚本（画面+字幕+配音）
- 设计 AI 视频风格和主体描述
- 生成首帧/尾帧的文生图提示词

## Seedance 2.0 核心概念

### 四种输入模态
| 类型 | 上限 | 用途 |
|------|------|------|
| 图片 | 9 张 | 角色形象、场景风格、首尾帧、商品外观 |
| 视频 | 3 个（≤15s） | 运镜参考、动作参考、转场效果 |
| 音频 | 3 个 MP3（≤15s）| 背景音乐、音色参考、节奏卡点 |
| 文字 | 无限制 | 画面描述、剧情、镜头指令 |

**总文件上限：12 个 | 生成时长：4-15 秒**

### @ 引用语法（必须！）
```
@图片1 作为首帧
@图片2 的人物作为主角
@视频1 参考镜头语言和运镜
```

**规则：素材不说清楚用途，模型会乱用！**

## 🎯 六棱镜法（必读！）

根据 @ponyodong 研究：角色一致性 = C + S + P + T + E + A

| 维度 | 含义 | 描述要点 |
|------|------|----------|
| **C** | Context 场景与角色 | 定妆照、表情、服装、人物特征 |
| **S** | Shot 镜头运动 | 起始镜头、运动路径、结束镜头 |
| **P** | Physics 物理反馈 | 服装飘动、发丝、光影反射、材质动态 |
| **T** | Time 时间线 | 总时长、各阶段秒数 |
| **E** | Editing 剪辑节奏 | 快慢、转场、定格 |
| **A** | Atmosphere 氛围 | 光线、色调、情绪 |

**自然语言流顺序：** 镜头运动 → 角色状态 → 物理反馈 → 光影氛围

❌ 分段列表（生成 10 次只能用 2-3 次）：
```
1. 特写→拉远
2. 主角走动
3. 长发飘动
4. 阳光照射
```

✅ 自然语言流（生成 10 次能用 7-8 次）：
```
镜头从特写缓慢拉远至全景，同时主角转身向右走动，长发自然向后飘动，阳光从左侧45度角照射，在地面形成长长的人影，画面色调为温暖的黄昏金色
```

## 🎬 分镜头脚本模板

**核心原则：使用自然语言流**

每个镜头的"画面"描述必须按照以下顺序：
> **镜头运动 → 角色状态 → 物理反馈 → 光影氛围**

**格式：** 每镜必须包含画面、字幕、配音三大要素

```
🎬 镜头 X（时间段）镜头简述

画面
[按顺序描述：镜头运动→角色/物体状态→物理细节→光影氛围]

字幕（大字）
[核心文案]

配音（语调）
[配音脚本]
```

### ❌ 错误格式（分段列表）
```
画面
产品居中
缓慢旋转
柔光灯照射
```

### ✅ 正确格式（自然语言流）
```

```

## 🖼️ 首帧提示词公式

```
[主体], [场景], [光影], [风格], [构图], [画质参数]
```

### 风格关键词速查

| 风格 | 关键词 |
|------|--------|
| 毛毡 | felt craft, wool texture, handmade, cozy, miniature |
| 商业摄影 | commercial photography, studio lighting, minimalist |
| 电影感 | cinematic, anamorphic, dramatic lighting |
| 赛博朋克 | cyberpunk, neon, holographic, rain-soaked |
| 胶片 | vintage, film grain, retro, analog |

### 镜头运动

| 中文 | 英文 | 效果 |
|------|------|------|
| 推 | push in / dolly in | 镜头靠近主体 |
| 拉 | pull out / dolly out | 镜头远离主体 |
| 环绕 | orbit shot | 绕主体旋转 |
| 俯拍 | overhead | 从上往下 |
| 跟拍 | follow shot | 跟随主体 |

### 一致性约束（加在提示词末尾）

```
Keep same character, same clothing, no face changes, high consistency.
```
