# Seedance 2.0 Prompt Generator

为 Seedance 2.0（字节跳动即梦 AI 视频模型）生成专业级提示词。

## 触发条件

当用户需要：
- 生成 Seedance 2.0 视频提示词
- 创建分镜头脚本（画面+字幕+配音）
- 设计 AI 视频风格和主体描述
- 生成首帧/尾帧的文生图提示词

## Seedance 2.0 核心概念

### 四种输入模态
| 类型 | 上限 | 用途 |
|------|------|------|
| 图片 | 9 张 | 角色形象、场景风格、首尾帧、商品外观 |
| 视频 | 3 个（≤15s） | 运镜参考、动作参考、转场效果 |
| 音频 | 3 个 MP3（≤15s）| 背景音乐、音色参考、节奏卡点 |
| 文字 | 无限制 | 画面描述、剧情、镜头指令 |

**总文件上限：12 个 | 生成时长：4-15 秒**

### @ 引用语法（必须！）
```
@图片1 作为首帧
@图片2 的人物作为主角
@视频1 参考镜头语言和运镜
```

**规则：素材不说清楚用途，模型会乱用！**

## 🎬 分镜头脚本模板

**格式：** 每镜必须包含画面、字幕、配音三大要素

```
🎬 镜头 X（时间段）镜头简述

画面
[环境][主体][动作][细节]

字幕（大字）
[核心文案]

配音（语调）
[配音脚本]
```

### 示例：产品展示

```
🎬 镜头 1（0-5s）产品旋转展示

画面
纯白无限背景，黑色机械键盘静置，RGB灯效缓慢呼吸

字幕
机械键盘

配音（冷静）
简约设计，RGB 灯效...
```

## 🖼️ 首帧提示词公式

```
[主体], [场景], [光影], [风格], [构图], [画质参数]
```

### 风格关键词速查

| 风格 | 关键词 |
|------|--------|
| 毛毡 | felt craft, wool texture, handmade, cozy, miniature |
| 商业摄影 | commercial photography, studio lighting, minimalist |
| 电影感 | cinematic, anamorphic, dramatic lighting |
| 赛博朋克 | cyberpunk, neon, holographic, rain-soaked |
| 胶片 | vintage, film grain, retro, analog |

### 镜头运动

| 中文 | 英文 | 效果 |
|------|------|------|
| 推 | push in / dolly in | 镜头靠近主体 |
| 拉 | pull out / dolly out | 镜头远离主体 |
| 环绕 | orbit shot | 绕主体旋转 |
| 俯拍 | overhead | 从上往下 |
| 跟拍 | follow shot | 跟随主体 |

### 一致性约束（加在提示词末尾）

```
Keep same character, same clothing, no face changes, high consistency.
```

展示 @图片1 产品，@图片2 参考材质。在纯白背景中央缓慢旋转 360°，镜头从中景推到特写。商业摄影风格，高级感，柔光灯。


## 完整工作流

1. **风格定位** → 2. **首帧生成** → 3. **分镜脚本** → 4. **Seedance 提示词**

```markdown
# [项目名]

## 🎨 风格定位
- **视觉：** [1-2 词]
- **比例：** [9:16 / 16:9]
- **时长：** [X 秒]

## 📋 分镜头
[按模板写 2-4 个镜头]

## 🎬 完整提示词
**素材：**
- 图片 1：[用途]
- 视频 1：[用途]（可选）

**提示词：**
[结合素材的分镜描述]
[风格约束 + 一致性要求]
```
