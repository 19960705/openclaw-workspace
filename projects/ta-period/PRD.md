# 「她周期」产品需求文档 (PRD)

## 一、产品概述

### 产品名
**她周期** | Period Log

### 一句话定位
帮助职场女性了解情绪周期与工作表现关联的数据工具。

### 核心价值观
- **不说教** — 只给数据，不给建议
- **不评判** — 情绪没有好坏，都是正常的
- **极简** — 只做一件事，做到极致

---

## 二、用户画像

### 核心用户
| 维度 | 描述 |
|------|------|
| 年龄 | 25-40 岁 |
| 职业 | 互联网/金融/外企 等高压行业的职场女性 |
| 痛点 | 情绪波动影响工作效率，但不知道为什么 |
| 诉求 | 了解自己，想优化工作状态 |
| 心理 | 不觉得自己"有病"，不需要"治疗" |

### 用户故事
> "每个月总有几天工作效率特别低，以前以为是自己的问题，现在知道了是周期规律，终于不再自责了。"

## 三、设计原则

### 1. 去医疗化
| 要 | 不要 |
|---|---|
| "那周"、"这周" | "经期"、"排卵期"、"黄体期" |
| "精力"、"能量" | "激素水平"、"雌激素" |
| "状态"、"感觉" | "症状"、"反应" |
| "周期规律" | "生理周期"、"月经周期" |

### 2. 不说教
| 要 | 不要 |
|---|---|
| "你这周能量偏低 📉" | "建议你这周多休息 😴" |
| "你上个月有 5 天状态不错 👍" | "根据你的数据，你应该..." |
| "这周工作效率平均提升 15%" | "你应该在这周安排重要会议" |

### 3. 纯工具，无疗愈
| 要 | 不要 |
|---|---|
| 纯数据记录 | 冥想引导音频 |
| 情绪可视化 | 心理咨询对话 |
| 周期曲线图 | "疗愈"、"治愈"文案 |
| 纯音乐背景（可选） | 心理测评 |

## 六、文案风格指南

### 语气
- 像闺蜜对话，不像医生
- 轻松、温暖、不评判
- 不用"应该"、"建议"、"必须"

### 用词对照

| 场景 | 推荐 | 避免 |
|------|------|------|
| 经期 | "那周"、"生理期" | "月经期"、"例假" |
| 排卵期 | 不出现 | "排卵期"、"安全期" |
| 精力 | "能量"、"🔋" | "激素"、"荷尔蒙" |
| 情绪 | "状态"、"感觉" | "症状"、"情绪化" |
| 低效日 | "效率偏低的日子的" | "生理期的" |
| 建议 | "你可以参考" | "你应该" |
| 结论 | "数据显示..." | "根据医学研究..." |

### 文案示例

| 场景 | 推荐 | 避免 |
|------|------|------|
| 周报开头 | "这周你辛苦了 💪" | "根据你的周期数据..." |
| 低效提醒 | "周三效率偏低，可能是那周来了" | "生理期会导致效率下降" |
| 状态好时 | "这周状态不错，效率提升了 20%" | "经期结束后状态回升" |
| 月度总结 | "你这个月有 8 天效率很高 👍" | "你这个月激素水平稳定" |
| 推送 | "快到那周了，记得对自己好一点" | "经期即将来临，请注意休息" |

### 数据结构

```typescript
interface DailyLog {
  date: string;           // 2026-02-15
  mood: 'happy' | 'okay' | 'tired' | 'crashed';
  energy: 1 | 2 | 3 | 4; // 🔋 数量
  efficiency: 'high' | 'medium' | 'low';
}

interface WeekSummary {
  weekNumber: number;
  moodDistribution: Record<Mood, number>;
  avgEnergy: number;
  efficiencyTrend: 'up' | 'down' | 'stable';
  insights: string[];
}
```
