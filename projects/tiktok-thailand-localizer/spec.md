# 泰国 TikTok 本地化文案适配工具

## 项目概述

一个基于1688产品链接，自动生成泰国TikTok电商高转化文案的本地化工具。

## 核心功能

### 1. 输入
- 1688产品链接
- （可选）目标国家：泰国（默认）/ 马来西亚

### 2. 分析
从1688链接获取：
- 产品标题
- 产品描述
- 产品图片
- 价格信息
- 类目信息
- 规格参数

### 3. 本地化适配

#### 3.1 产品标题生成（3种公式）

**公式 A: 强搜索型**
```
{品类词} + {关键特征1/2} + {数量/规格} + {场景/收益}
```

**公式 B: 强促销型**
```
{โปร/优惠} + {品类} + {关键收益/保证}
```

**公式 C: 成分/功效型**
```
{产品形态} + {主成分} + {功效} + {规格}
```

#### 3.2 详情页文案生成（5模块结构）

```
【一句话定位】
→【3-5条卖点】
→【规格与包含物】
→【使用方法/适用人群】
→【信任与保障】
```

#### 3.3 视频/直播文案生成（4段式结构）

```
【情绪钩子 Hook】
→【交易利益 Offer】
→【确定性承诺 Trust】
→【极短 CTA（购物篮）】
```

#### 3.4 自动添加

**高频转化词:**
- `คุ้มๆ / คุ้มสุด`（划算/超值）
- `1แถม1`（买一送一）
- `ไม่ผิดหวังแน่นอน`（保证不失望）
- `ติดแน่น`（很牢）
- `พิกัดในตะกร้า`（链接在购物篮）⭐⭐⭐⭐⭐

**Hashtag策略:**
```
#类目词 + #场景词 + #平台词
```

#### 3.5 文化避坑扫描

自动检查并警告：
- 紫色（泰国哀悼色）
- 宗教符号误用
- 敏感政治话题

### 4. 输出

**产品卡片:**
```
┌─────────────────────────────┐
│ 1688产品                  │
│ - 标题                    │
│ - 主图                    │
│ - 价格                    │
└─────────────────────────────┘
```

**本地化文案:**
```
┌─────────────────────────────┐
│ 🇹🇭 泰国 TikTok 文案      │
├─────────────────────────────┤
│ 1. 产品标题               │
│    [公式A] [公式B] [公式C] │
├─────────────────────────────┤
│ 2. 详情页文案             │
│    [5模块结构]             │
├─────────────────────────────┤
│ 3. 视频/直播文案          │
│    [4段式结构]             │
├─────────────────────────────┤
│ 4. Hashtag               │
│    [类目+场景+平台]        │
└─────────────────────────────┘
```

**检查清单:**
```
✅ 第一句是Hook？
✅ 有交易利益？
✅ 有确定性词？
✅ CTA明确到购物篮？
✅ Hashtag完整？
```

## 技术架构

### 前端

**技术栈:**
- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- Shadcn/ui (组件库)
- TypeScript

**页面结构:**
```
/landing          - 产品介绍
/app
  /             - 主界面
    /analyze     - 分析结果页
```

### 后端

**API Routes:**
```
/api/analyze      - 分析1688产品并生成文案
/api/download     - 下载文案（JSON/Markdown）
/api/history      - 历史记录
```

**核心服务:**
```
services/
  ├── 1688-scraper.ts      - 1688产品爬取
  ├── translation.ts       - 中→泰翻译
  ├── template-engine.ts   - 模板引擎
  ├── keyword-generator.ts - 关键词生成
  ├── hashtag-generator.ts- Hashtag生成
  └── culture-scanner.ts  - 文化避坑扫描
```

**数据源:**
```
data/
  ├── templates/
  │   ├── titles.json      - 标题模板
  │   ├── descriptions.json - 详情页模板
  │   └── videos.json     - 视频/直播模板
  ├── keywords/
  │   ├── conversion.json  - 转化词（泰语）
  │   ├── categories.json - 类目词
  │   └── scenarios.json  - 场景词
  └── culture/
      └── taboos.json     - 避坑词
```

### 翻译方案

**优先级:**
1. **AI翻译**（Claude API / OpenAI）- 主要方案
2. **Google Translate API** - 备选
3. **本地词典** - 特定词汇直译

**翻译策略:**
- 核心术语（品类词、转化词）用本地词典
- 描述性文案用AI翻译
- 人工review关键结果

### 部署

**开发环境:**
- Vercel（推荐）- 快速部署Next.js
- 或 Netlify

**生产环境:**
- Vercel Pro
- 域名: thailocalizer.com（示例）

## 风险与挑战

### 技术风险

1. **1688反爬机制**
   - 解决: 使用代理池、限制请求频率、备用方案（手动输入产品信息）

2. **翻译质量**
   - 解决: AI翻译 + 人工review、用户反馈优化、本地词典

3. **模板局限性**
   - 解决: 持续收集热销样本、A/B测试优化模板

### 业务风险

1. **市场接受度**
   - 解决: 先做MVP验证、小范围测试、用户反馈快速迭代

2. **竞争者**
   - 解决: 专注泰国市场、本地化深度、转化率导向

3. **盈利模式**
   - 解决: 免费版吸引用户 + 付费版解锁高级功能
