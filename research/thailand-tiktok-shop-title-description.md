## 1. TikTok Shop 商品页文案字段（可控位）

### 1.1 产品链接标题（Product Title）

**出现位置:**
- 商品卡片列表标题
- PDP 主标题（通常为 H1）
- 外链分享的页面标题 / SEO 标题（标题过长会被截断）

### 1.2 变体/规格命名（Variant / Model / Color / Size）

**示例:** `Rechargeable Pump(Gray)+ 9 Bags`

### 1.3 详情页"硬信息区"（Details / Specifications）

**结构化字段（属性名 + 属性值）:**
- Closure Type（闭合类型）
- Material（材质）
- Feature（特点）
- Place（使用场景）

### 1.4 详情页"软说服区"（Product description）

**最关键的"文案区"，决定用户是否相信、是否愿意下单**

**常见形态:**
1. 1 句总括（产品是什么/解决什么）
2. 分段卖点（带 emoji 或符号的 bullet）
3. 数据化参数（电池、功率、尺寸、时间等）
4. 使用场景（旅行、日常、送礼等）

### 1.5 信任构件（Trust blocks）

- Reviews（用户评价）
- Shipping & returns（运费/退货）
- TikTok Shop protections（平台保障）
- About this shop（店铺星级、发货时效、好评率等）

## 2. 可访问样本分析

### 样本 A（家居收纳/旅行压缩袋）

**链接:**
https://www.tiktok.com/shop/pdp/vacbird-travel-vacuum-bags-9-pack-with-rechargeable-air-pump/1730840089017160145

**标题（节选）:**
`VACBIRD [2026 Upgraded] X7 Cordless Travel Vacuum Bag Set ... 15 Pack ... Space Saver ...`

**Details（结构化属性）:**
```
Closure Type: Zip
Occasion: Daily,Business
Material: PE
Feature: Recyclable,Waterproof
Storage Type: With Zip
Place: Bedroom
```

**Product description（节选）:**
```
VACBIRD X7 Cordless Travel Vacuum Bag Set

✨ Powerful Compression: ... 1500mAh ... 4000Pa ... 10-15 seconds ...
Includes 9 Vacuum Bags

📦 3 Sizes: ... reduce luggage volume by up to 80% ...
```

**关键发现:**
- "Details"负责可比价的**标准参数**
- "Product description"负责让人买单的**价值叙事 + 数据证明**
- 文案分段明显，且常用 emoji 作为视觉锚点

## 3. 产品链接标题：高转化标题的 6 个特征

### 3.1 以"品类词"开头，先让系统/用户识别你卖什么

**示例:** `Vacuum Bag Set / Phone Case / Serum / Snack Gift Set`

### 3.2 关键词堆叠，但要有"层级"

**推荐层级:**
1. **品牌/系列（可选）**: VACBIRD / Dr.Melaxin
2. **核心品类**: Travel Vacuum Bag Set
3. **关键特征 2-4 个**: Cordless / Rechargeable / Waterproof
4. **数量/规格**: 15 Pack / 500ml / 30 pcs
5. **使用场景/收益**: Space Saver / For Luggage

### 3.3 用"数字"制造确定性

| 类型 | 示例 |
|------|------|
| 数量 | 15 Pack / 9 Bags / 3 Sizes |
| 性能 | 4000Pa / 1500mAh |
| 时间 | 10-15 seconds |

### 3.4 标题里放 1 个"强利益点"而不是全是名词

**示例:**
- `Space Saver`（省空间）
- `Long-lasting`（更耐用）
- `Strong magnetic`（磁吸牢）

### 3.5 避免无意义形容词堆砌

**示例:**
- ❌ 过多的 `Amazing / Best / Perfect` 对搜索与理解贡献不大
- ✅ 更有效的是"可验证的具体信息"

### 3.6 泰国站本地化：标题最好"泰语为主 + 英文补充"

**原因:**
- 泰国用户浏览与搜索主要是泰语
- 但部分品类词（3C/美妆成分）英文更通用

## 4. 详情页文案：高转化写法的 5 个模块

### 模块 1: 一句话定位（这是什么 + 解决什么）

**形式:** `产品名 + 核心收益`

**泰语模板:**
- `เซ็ต{品类} สำหรับ{场景} ช่วย{核心收益}`
- `ตัวนี้ช่วย{痛点解决} ใช้ง่ายมาก`（偏口语）

### 模块 2: 3–5 条卖点（每条：功能 → 结果 → 证据）

**强烈建议每条都带:**
- **结果词**: ประหยัดเวลา/ประหยัดพื้นที่/ติดแน่น/ไม่ระคายเคือง
- **数字或规格**: mAh、Pa、ขนาด、จำนวน

### 模块 3: 规格与包含物（What's in the box）

**在泰国快消/礼盒特别重要:**
- `ได้อะไรบ้าง / มีอะไรในเซ็ต`

### 模块 4: 使用方法/适用人群（How to use / For who）

**解决"我会不会用/适不适合我"的犹豫**

### 模块 5: 信任与保障（呼应平台/店铺信任模块）

**泰国高频确定性词:**
- `ตรงปก`（和图一致）
- `ไม่ผิดหวังแน่นอน`（不失望）
- `ส่งไว`（发货快）
- `ของแท้`（正品）

## 5. 泰国站可直接套用的标题公式

### 公式 A: 强搜索型（适合大多数类目）

**结构:**
`{品类词} + {关键特征1/2} + {数量/规格} + {场景/收益}`

**泰语示例（手机壳）:**
```
เคสไอโฟน แม่เหล็ก Magsafe ติดแน่น หมุนได้ 360° + Griptok (เซ็ต 2 ชิ้น)
```

**泰语示例（零食礼盒）:**
```
ตะกร้าของขวัญปีใหม่ เซ็ตขนมดังในติ๊กต๊อก + การ์ดอวยพร (เลือกขนมได้)
```

### 公式 B: 强促销型（更像"活动标题"）

**结构:**
`{โปร/优惠} + {品类} + {关键收益/保证}`

**泰语示例:**
```
โปร 1แถม1 เคสแม่เหล็กไอโฟน + Griptok คุ้มสุด ติดแน่น
```

### 公式 C: 成分/功效型（适合美妆/保健）

**结构:**
`{产品形态} + {主成分} + {功效} + {规格}`

**泰语示例:**
```
เซรั่มวิตามินซี ลดรอยดำ ผิวใส 30ml (อ่อนโยน ไม่ระคายเคือง)
```
