# 自我进化记录

## 每周自检清单

### 1. 能力盘点
- [ ] 这周学到了什么新技能？
- [ ] 有没有重复解决以前解决过的问题？
- [ ] 有哪些经验可以写成 skill？

### 2. 效率检查
- [ ] 工具调用有没有可以优化的？
- [ ] 有没有更高效的方式完成常见任务？

### 3. 记忆检查
- [ ] 学到的重要东西是否都存到 MEMORY.md 了？
- [ ] 有没有忘记之前记录的重要信息？

### 4. 主动性检查
- [ ] 有没有主动发现问题/机会？
- [ ] 有没有主动推进项目？

---

## 进化触发条件

满足任一即触发深度自检：
- 连续 3 次用同一种方式解决类似问题
- 用户反馈"这个之前做过"
- 发现可以复用的工具/流程

## 产出要求

每次自检必须产出：
1. 至少 1 条可复用的经验 → 写入 MEMORY.md
2. 至少 1 个可优化的流程 → 考虑写成 skill/hook
