## 提高转化率的小技巧

1. **展示样品** - 把自己最好的作品放到 Gig 里
2. **快速响应** - Fiverr 算法喜欢回复快的卖家
3. **积累评价** - 前几单可以低价做，建立口碑
4. **视频介绍** - 添加自我介绍视频增加信任
