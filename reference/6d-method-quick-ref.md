# 🎯 六棱镜法 Quick Reference

## 核心公式
```
角色一致性 = C + S + P + T + E + A
```

## 6D 检查清单

| 维度 | 问题 | 检查点 |
|------|------|--------|
| **C** | 角色是谁？ | 定妆照、表情、动作、服装、品牌元素 |
| **S** | 镜头怎么动？ | 起始→运动路径→结束，特写/全景/环绕 |
| **P** | 物理细节？ | 服装飘动、发丝、光影反射、材质感 |
| **T** | 时间分配？ | 总时长、各阶段秒数、节奏分布 |
| **E** | 剪辑节奏？ | 快慢、转场、定格、情绪点 |
| **A** | 氛围调性？ | 光线、色调、情绪、视角 |

## 自然语言流顺序
> 镜头运动 → 角色状态 → 物理反馈 → 光影氛围

## ⚠️ 常见翻车原因
- 只给参考图，没给"怎么动" → 脸会变
- 分段列表 → 生成 10 次只能用 2-3 次
- 光影不统一 → 同一集画面忽明忽暗

## ✅ 正确 vs ❌ 错误

✅ 自然语言流：
> 镜头从特写缓慢拉远至全景，同时主角转身向右走动，长发自然向后飘动，阳光从左侧45度角照射，在地面形成长长的人影，画面色调为温暖的黄昏金色。

❌ 分段列表：
> 1. 特写→拉远
> 2. 主角走动
> 3. 长发飘动
> 4. 阳光照射
