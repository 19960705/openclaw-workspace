# AI 视频生成工具对比研究 (2026年2月)

## 三大工具概览

### 1. Perplexity
- **定位**: 研究/工作流助手内嵌视频生成
- **视频时长**: 最长 8 秒
- **订阅要求**: Pro / Max / Enterprise
- **特色功能**: 
  - 上传图片作为首帧
  - 无法编辑已生成的视频（只能重新生成）
- **定价**: 按月分配额度，30天滚动更新，无法购买额外积分

### 2. Flowith (目黒琉璃)
- **定位**: 积分制多模型创作工作区
- **订阅**: Ultimate (已登录)
- **视频模型**: Veo 3.1, Kling, Sora 等
- **定价**:
  - Pro: $15.32/月 (22,000 积分)
  - Ultimate: $39.92/月 (85,000 积分)
  - Infinite: $459.90/月 (1,000,000 积分)
- **特色**: 
  - Agent Mode 自主规划
  - 支持文档、幻灯片、网页、游戏、视频、图片多种输出
  - 知识库功能

### 3. Jimeng (即梦) - Seedance 2.0
- **定位**: 字节跳动旗下专业 AI 视频生成
- **分辨率**: 最高 2K
- **特色功能 Seed**:
  -ance 2.0 全能参考模式
  - Agent 模式
  - 图片生成、视频生成、数字人、动作模仿
- **适用场景**: 中文提示词理解更强

## 快速使用 Skill

### Perplexity 搜索
```
1. browser open url=https://www.perplexity.ai
2. 点击 New Thread
3. 输入搜索查询
4. 按 Enter 搜索
```

### Flowith 图像/视频生成
```
1. browser open url=https://flowith.io
2. 点击 新对话
3. 在输入框描述需求
4. 选择输出类型 (图片/视频/文档等)
5. 点击开始
```

### Jimeng/Seedance 视频生成
```
1. browser open url=https://jimeng.jianying.com
2. 选择 Agent 模式/视频生成
3. 输入提示词或上传参考图
4. 点击生成
```

## 推荐工作流

1. **市场调研** → Perplexity 搜索
2. **产品图生成** → Flowith 图片生成
3. **视频制作** → Jimeng Seedance 2.0
4. **批量处理** → Flowith Agent Mode
