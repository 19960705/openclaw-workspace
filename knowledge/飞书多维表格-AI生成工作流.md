# 飞书多维表格 AI 生成工作流

## 爆款内容自动化工厂

**参考文章**: [多维表格实战教程：如何搭建爆款内容自动化工厂？](https://www.feishu.cn/content/article/7584012383060757728)

### 完整工作流

1. **📥 抓取** → 抖音/小红书/B站数据
   - 插件市场搜"抖音数据小助手"
   - 10篇帖子2秒抓取，效率提升900倍

2. **📊 分析** → AI字段自动分析
   - 内容结构分析
   - 受众分析
   - 文案结构拆解
   - 产品卖点分析

3. **✍️ 仿写** → 批量生成文案
   - 违禁词自动检测
   - 跨平台内容转换

4. **🎨 生图** → 即梦/豆包/Nano Banana

5. **🎬 生视频** → 即梦 4.0

6. **📈 仪表盘** → 数据闭环优化

## AI 生图/视频 专用

**参考文章**: [多维表格AI字段捷径：如何用AI实现批量文生图和文生视频？](https://www.feishu.cn/content/article/7592538064711470271)

### 支持的模型

| 模型 | 用途 |
|------|------|
| **即梦 4.0** | 文生图、图生图、4K超清 |
| **豆包** | 海报生成、头像特效 |
| **Nano Banana Pro** | 专业级图像生成 |
| **Vidu** | 图生视频，最快10秒 |

### 快速开始

1. 打开飞书多维表格
2. 右键 → 插入字段捷径 → AI字段
3. 选择"图像生成"或"视频生成"
4. 输入提示词，批量生成！
