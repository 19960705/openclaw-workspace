# 赏金任务平台指南

## 平台对比

| 平台 | 网址 | 特点 | 难度 |
|------|------|------|------|
| **Algora** | algora.io | 最大，代码任务多 | 中高 |
| **Opire** | opire.dev | 任何人可创建赏金 | 初学者友好 |
| **Gitcoin** | gitcoin.co | Web3 + 赏金 | 中 |
| **HackerOne** | hackerone.com | 安全漏洞 | 高 |

## 起步指南

### 1. 注册账号
- 用 GitHub 账号登录（Algora、Opire 都支持）
- 需要 Stripe 收款（国外）
- Opire：reward 从 $20 开始

### 2. 找简单任务
- 搜索 "good first issue"
- 小额赏金 $5-$100
- 文档改进、测试用例、简单 Bug

### 3. 接单流程
1. 在 issue 下评论 `/claim`
2. 提交 PR
3. PR 合并后收款

### 4. 新手建议
- 从小额开始 ($5-$50)
- 选熟悉的语言/框架
- 先看项目贡献指南
