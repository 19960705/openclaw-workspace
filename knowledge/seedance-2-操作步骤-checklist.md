
# Seedance 2.0 操作步骤 Checklist

## 准备工作
- [ ] 在 Chrome 浏览器里点击 OpenClaw 扩展图标，连接一个 tab（profile "chrome"）
- [ ] 确认浏览器服务正常（`browser action=status`）
- [ ] 准备主体参考图（用 Google Flow 生成或产品原图）
- [ ] 确认即梦账号已登录

---

## Step 1: 生成主体参考图（AI Master 三步法 - Step 1）
- [ ] 打开 Google Flow（labs.google.com/fx/flow）
- [ ] 写提示词，锁死角色、服装、场景、光线、风格
- [ ] 生成主镜头图像（这步不满意 → 后面全返工！）
- [ ] 保存参考图到本地

---

## Step 2: 上传参考图到即梦
- [ ] 打开即梦（jimeng.jianying.com/ai-tool/home）
- [ ] 进入"视频生成"
- [ ] 点击"上传"按钮
- [ ] 选择准备好的主体参考图
- [ ] 等待上传完成

---

## Step 3: 选择主体参考模式
- [ ] **重要！** 选择"主体参考"模式（不是"首帧"！）
- [ ] 在提示词中用 @ 引用图片：
  ```
  @图片1 作为产品主角，保持不锈钢量杯的外观和材质不变
  ```
- [ ] 说明清楚参考图的用途（不然模型会乱用！）

---

## Step 4: 生成分镜头提示词
- [ ] 用 seedance-prompt-generator 生成高质量分镜提示词
- [ ] 或者自己写分镜头脚本（10-15秒视频需要多个镜头）
- [ ] 分镜结构示例：
  ```
  镜头 1（0-3s）：[画面描述] + [字幕] + [配音]
  镜头 2（3-6s）：[画面描述] + [字幕] + [配音]
  镜头 3（6-10s）：[画面描述] + [字幕] + [配音]
  ```

---

## Step 5: 生成视频
- [ ] 选择模型：**Seedance 2.0**（第一个选项）
- [ ] 选择时长：5s/10s/15s（建议先 5s 测试）
- [ ] 点击生成按钮
- [ ] 等待视频生成完成
- [ ] 检查效果，确认主体一致性

---

## Step 6: 多片段生成长视频（突破 15 秒限制）
- [ ] 把长视频拆成多个 ≤15s 片段
- [ ] 每个片段都用相同的主体参考图
- [ ] 用上一段的尾帧作为下一段的首帧
- [ ] 保持角色一致性、光线一致性、色调一致性
- [ ] 用剪映等工具拼接

---

## Step 7: 保存和记录
- [ ] 保存生成的视频到"资产"
- [ ] 记录积分消耗
- [ ] 记录哪些提示词效果好，哪些不好
- [ ] 更新学习笔记

---

## 常见问题排查
- **浏览器服务不可用**：在 Chrome 里点击 OpenClaw 扩展图标连接 tab（profile "chrome"）
- **主体参考模式 vs 首帧模式**：一定要选"主体参考"，不是"首帧"！
- **提示词效果不好**：用 @ 引用图片，说明清楚用途
- **片段衔接不自然**：保持主体、光线、色调全程一致
