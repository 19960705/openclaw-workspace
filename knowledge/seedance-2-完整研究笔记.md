## 一、完整操作流程

### 1.1 生成参考图（可选）
1. 打开即梦 → 图片生成
2. 输入提示词（锁死角色、服装、场景、光线、风格）
3. 选择"图片 4.1"模型
4. 点击生成

### 1.2 生成视频
1. 打开即梦 → 视频生成
2. 选择模型：**Seedance 2.0**（全能王者）
3. 选择模式：**全能参考**
4. 选择比例和时长：3:4, 15s
5. 上传参考图（两种方式）：
   - **方式一**：点击已生成的图片 → 点"生成视频" → 自动添加到参考内容
   - **方式二**：点击"参考内容"上传区 → 从本地上传
6. 在提示词里用 @图片1 引用参考图
7. 点积分旁边的黑色圆形按钮提交

---

## 二、上传参考图的正确方法

### 方法一：从已生成的图片添加（推荐）
1. 在生成列表里点击一张已生成的图片
2. 图片会打开大图预览
3. 点"生成视频"按钮
4. 图片自动添加到参考内容，提示词自动变成 @图片1 xxx
5. 修改提示词，保留 @图片1 引用

### 方法二：从本地上传
1. 点击"参考内容"上传区
2. 拖拽图片到上传框
3. 在提示词里用 @图片1 引用

## 三、编辑功能

### 3.1 对口型 (Lip Sync)
- 输入文字让角色说话
- 上传音频文件
- 数字人模式
- 快速模式

### 3.2 AI音效 (AI Sound Effects)
- 画面内容分析：自动分析视频，生成匹配声音
- 音效描述：手动描述想要的音效
- 风格：自然、写实
- 示例：海浪声、海鸥鸣叫、脚步声

### 3.3 AI配乐 (AI BGM)
- 根据视频内容生成背景音乐

### 3.4 补帧 (Frame Interpolation)
- 提升视频流畅度

### 3.5 提升分辨率 (Upscale)
- 提高视频分辨率

## 四、注意事项

### 4.1 审核问题
- 避免提示词里出现"17岁"、"未成年"、"学生"等关键词
- 容易触发内容审核导致视频未通过

### 4.2 浏览器问题
- 使用 profile "openclaw" 无头浏览器
- 即梦的 dialog 有时会遮挡按钮，需要先关闭

## 六、踩坑记录

### 6.1 审核敏感词
- ❌ "17岁高中女生" → 触发审核
- ✅ "少女" "年轻女孩" → 安全
- 中国平台对未成年人内容审核极严

### 6.2 切换模式会重置模型
- 选好 Seedance 2.0 后切换"全能参考"，模型会回退到 Fast
- 正确顺序：先选模式 → 再选模型

### 6.3 上传参考图
- browser upload 工具对即梦不可用（路径验证 + React 控制组件）
- 最佳方式：从已生成图片点击"生成视频"自动添加参考

### 6.4 查看结果
- 生成页滚动不可靠（自定义滚动容器）
- 直接去"资产→视频"页面查看更稳定

## 七、最佳实践

1. 提示词不写年龄、不写学生
2. 操作顺序：模式 → 模型 → 比例/时长 → 提示词
3. 参考图用"已生成图片→生成视频"方式添加
4. 浏览器超时就 stop → start → open 三步重启
5. 查看结果去"资产→视频"页面
6. 审核不通过不扣积分，可以放心尝试
