# Model Context Protocol (MCP) 2026

> 来源: Anthropic, Wikipedia, Red Hat
> 日期: 2025-2026

## 什么是 MCP?

Model Context Protocol (模型上下文协议) 是一个开放标准，让 AI Agent 可以：

1. **动态发现工具** - 无需重新编程就能使用新工具
2. **跨平台调用** - 统一接口调用不同服务的工具
3. **上下文共享** - Agent 之间可以共享上下文

## 谁在用?

| 公司 | 用途 |

| **Block** | 金融数据 |

| **Codeium** | 代码补全 |
| **Sourcegraph** | 代码搜索 |
| **Red Hat** | OpenShift AI |

## MCP vs 传统 API

| 传统 API | MCP |
|----------|-----|
| 固定接口 | 动态发现 |
| 需编程集成 | 声明式配置 |
| 单一服务 | 跨服务调用 |

## 安全问题 (2025.4)

- Prompt injection
- 工具权限组合导致数据泄露
- 仿冒工具替换信任工具
