## 🎯 铺货策略建议

### 产品矩阵策略

**三层产品结构**:

1. **引流款 (30-50元)**
   - 目标: 获取流量，建立认知
   - 推荐: RAZO、Withat 类型
   - 数量: 2-3个SKU
   - 利润率: 20-30%

2. **主力款 (80-120元)**
   - 目标: 主要销量和利润来源
   - 推荐: LACANA、Madame Fin 类型
   - 数量: 3-5个SKU
   - 利润率: 40-50%

3. **利润款 (120-200元)**
   - 目标: 提升品牌形象，高利润
   - 推荐: Love Potion、Magic Spell 类型
   - 数量: 2-3个SKU
   - 利润率: 50-60%

### 套装组合策略

**高成交套装模式**:

1. **买1送1**: 2瓶同款或不同款
2. **多件套装**: 香水 + 乳液/护肤品
3. **赠品套装**: 香水 + 镜子 + 钥匙扣 + 布袋
4. **买2送赠品**: 购买2件送小饰品

**建议**:
- 每个SKU设计2-3种套装
- 套装价格比单买优惠15-20%
- 赠品成本控制在5-10元

### 定价策略

**定价公式**:
```
成本价 x 2.5 - 3 = 售价
```

**价格锚点**:
- 引流款: ¥39-49 (心理价位50以下)
- 主力款: ¥79-99 (心理价位100以下)
- 利润款: ¥129-149 (心理价位150以下)

**促销策略**:

- 满减活动: 满2件减10元
- 新客优惠: 首单立减15元
