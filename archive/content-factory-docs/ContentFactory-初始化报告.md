## 🎯 使用指南

### 快速开始

1. **捕获灵感**
   ```
   发送链接或文本给 Keonho
   → 自动保存到收件箱
   → 分类到对应灵感库
   → 生成选题
   ```

2. **开始创作**
   ```
   "今天写什么？"
   → 推荐选题
   → 生成大纲
   → 打磨初稿
   → 多平台适配
   ```

3. **发布归档**
   ```
   "已发布"
   → 移动到归档
   → 记录数据
   → 复盘总结
   ```

### 特殊指令

- `整理收件箱` - 批量处理待分类素材
- `推荐今日选题` - 智能推荐选题
- `检查进行中的选题` - 查看未完成任务
- `生成本周内容报告` - 数据统计

## ⚠️ 注意事项

1. **文件操作**
   - 移动文件使用 `mv` 命令
   - 保持文件名不变
   - 重命名需谨慎

2. **双向链接**
   - 使用 `[[文件名]]` 格式
   - 不要包含路径和扩展名
   - 文件名要有辨识度

3. **元数据**
   - 所有文件必须包含 YAML
   - 创建时自动生成
   - 更新时同步修改

4. **备份**
   - 定期备份 ContentFactory 目录
   - 可以考虑用 Git 管理
   - 排除敏感信息
