# TikTok 泰区内容管理系统 - 飞书多维表格配置文档

## 📋 系统概述

**目标**：搭建一个自动化的 TikTok 泰区爆款内容抓取、分析、仿写、发布、复盘的完整系统

**核心功能**：
- 自动抓取泰区热门视频
- AI 分析爆款规律
- 批量仿写内容
- 数据追踪与复盘
- 趋势监控与选品建议

---

## 🤖 自动化配置

### 自动化1：新增视频自动触发分析

**触发条件**：表1新增记录时

**执行动作**：
1. 在表2中创建关联记录
2. 触发所有AI字段分析

**配置路径**：表1 → 自动化 → 新建自动化 → 当记录创建时 → 创建关联记录

---

### 自动化2：每日数据更新

**触发条件**：每天 23:00

**执行动作**：
1. 调用 TikTok API 更新表4的播放/点赞/评论数据
2. 自动计算 ROI
3. 自动勾选"是否爆款"

**配置路径**：表4 → 自动化 → 新建自动化 → 定时触发 → 每天23:00

---

### 自动化3：爆款提醒

**触发条件**：表4中"是否爆款"被勾选时

**执行动作**：
1. 发送飞书消息通知团队
2. 消息内容：视频链接 + 播放量 + ROI

**配置路径**：表4 → 自动化 → 新建自动化 → 当字段更新时 → 发送消息

## 🔧 泰区特色配置

### 1. 泰文-中文双语支持

所有涉及泰文的字段，都配置对应的"中文翻译"AI字段，方便团队审核。

### 2. 本地化适配检查

在表2的"泰区适配建议"AI字段中，特别强调：
- 泰国文化禁忌（如皇室、宗教）
- 节日/热点事件（如泼水节、国王生日）
- 当地消费习惯

### 3. 泰铢成本核算

所有涉及金额的字段，统一使用泰铢（THB）作为单位。

### 4. 泰国时区优化

- 发布时间建议：泰国时间 18:00-22:00（黄金时段）
- 定时任务按泰国时区（UTC+7）配置

## 📝 使用流程

### 第一步：抓取爆款内容

1. 手动或通过工具抓取泰区热门视频
2. 填入表1：视频链接、标题、文案、数据等
3. 系统自动触发表2的AI分析

### 第二步：筛选优质内容

1. 查看表2的分析结果
2. 筛选"可复制性评估"为"高"的内容
3. 查看"爆款因素评分"总分>15的内容

### 第三步：批量仿写

1. 在表3中创建新记录，关联到表1的优质内容
2. 系统自动生成仿写标题和文案
3. 检查"违禁词检测"结果
4. 手动调整和完善

### 第四步：制作发布

1. 根据表3的拍摄脚本制作视频
2. 更新"制作状态"
3. 按计划发布时间发布
4. 在表4中记录发布链接

### 第五步：数据追踪

1. 系统每天23:00自动更新表4的数据
2. 查看仪表盘了解整体表现
3. 对爆款内容进行复盘

### 第六步：趋势监控

1. 每天在表5中记录当日趋势
2. 查看"选品建议"AI字段
3. 调整下一步的内容方向

## ⚠️ 注意事项

1. **AI字段需要手动配置**：创建表格后，需要在飞书界面手动添加AI字段并设置提示词

2. **API权限**：如果需要自动抓取数据，需要申请 TikTok API 权限

3. **数据更新频率**：建议每天更新一次数据，避免频繁调用API

4. **成本控制**：AI字段会消耗飞书AI额度，建议合理使用

5. **团队协作**：建议设置不同角色权限：
   - 运营：可编辑所有表格
   - 制作：只能编辑表3和表4
   - 管理：可查看所有数据和仪表盘
